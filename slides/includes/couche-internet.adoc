== Adresse IP

* "I.P." == "Internet Protocol"
* Adresse logique (virtuelle)

== Adresse IP : Format

* Taille de 32 bits (4 octets)
* Exprimé avec 4 nombres décimaux
+
```
0.0.0.0
...
10.0.3.4
...
192.168.0.1
...
255.255.255.255
```

== Adresse IP : Codage

L'adresse IP porte 2 informations:

* L'adresse de l'hôte
* L'adresse du réseau
